# Final Project: Interactive Weather App (Python Basics)
  Developed a weather application using Python while building a strong foundation in core programming concepts. The app processes user input to display current temperature in both Celsius and Fahrenheit using string handling and data conversion. Using conditional statements and functions to organize and process weather data, as well as Python date and time features to ensure accurate, developed a final project using real-time forecasts through Weather API integration.

  With my background in special education and a focus on accessibility, I designed the interface around clear user input and readable forecast output using the Rich library. High-contrast color combinations such as blue, yellow, and white were applied to improve readability for users with color blindness. Integratring live weather data and with a structured the output improved clarity and usability. While the application can be further improved, it represents my beginner stage in learning a new programming language and demonstrates my growth in Python development and problem-solving.
